## Requirements:

### Python 
- https://www.python.org/downloads/ 

### Pip Package Manager
- https://pip.pypa.io/en/stable/installation/ 
- `python get-pip.py`

### Django 
- `python -m pip install Django`

## How to run local server:
- `cd mysite`
- `python manage.py runserver`
- http://127.0.0.1:8000
